# About

This repository is the API for the [uFilterPickerPickProxy](https://github.com/uofuseismo/uFilterPickerPickProxy) - a utility that aggregates and deduplciates uFilterPick picks from various instances running the [uFilterPicker](https://github.com/uofuseismo/uFilterPicker) software then makes those picks available to downstream consumers in the UUSS Kubernetes system.
